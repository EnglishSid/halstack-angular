Properties

Name Default Description selected: boolean false If true, the component is selected. If undefined, the component will be uncontrolled and the value will be managed internally by the component. mode: string ('basic' | 'outlined') 'basic' Uses one of the available toggle modes. iconPosition: string ('before' | 'after') 'before' Changes the position of the icon. theme: string ('light' |'dark') 'light' Uses one of the available component themes. label: string Text to be placed on the toggle. iconSrc: string URL of the icon that will be placed in the toggle. disabled: boolean false If true, the component will be disabled. disableRipple: boolean 'false' If true, the ripple effect will be disabled. onClick: EventEmitter This function will be called when the user clicks the toggle. The new value will be passed as a parameter. margin: any (string | object) Size of the margin to be applied to the component ('xxsmall' | 'xsmall' | 'small' | 'medium' | 'large' | 'xlarge' | 'xxlarge'). You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different padding sizes. size: any (string | object) 'fitContent' Size of the component ('small' | 'medium' | 'large'| 'fitContent' | 'fillParent').
