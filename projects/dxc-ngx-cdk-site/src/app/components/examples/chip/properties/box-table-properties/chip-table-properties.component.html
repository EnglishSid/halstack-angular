Properties

Name Default Description label: string Text to be placed inside the chip. suffixIconSrc: string Path of the icon to be placed after the label. prefixIconSrc: string Path of the icon to be placed before the label. margin: any (string | object) Size of the margin to be applied to the component ('xxsmall' | 'xsmall' | 'small' | 'medium' | 'large' | 'xlarge' | 'xxlarge'). You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different margin sizes. disabled: boolean false If true, the component will be disabled. backgroundColor: string Color that will be applied to the background of the chip. borderColor: string Color that will be applied to the solid border of 1 px. suffixIconClick: EventEmitter Event that will be emitted when the suffix icon is clicked. prefixIconClick: EventEmitter Event that will be emitted when the prefix icon is clicked.
